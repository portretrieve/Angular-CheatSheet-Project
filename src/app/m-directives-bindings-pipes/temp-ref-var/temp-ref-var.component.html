Template Refrence Variables

Template reference variable is used to create a variable of any DOM Element like P Tag, Div Tag, Input Element, Form Element and more. Which Can be accessed anywhere in DOM.

Input Element

Variable (#myInput): {{ myInput }}

Value (myInput.value): {{ myInput.value }}

Paragraph Element

Hello There

Variable (#myPara) : {{ myPara }}

Textcontent (myPara.textContent): {{ myPara.textContent }}

Form Element

Variable (#myForm) : {{ myForm }}

Textcontent (myForm.textContent): {{ myForm.textContent }}
